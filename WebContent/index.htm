Real-Time Social Experiments in the Browser

nodeGame is a free and open source javascript framework for online, multiplayer, real-time games and experiments in the browser.

Get started

  1. Download the latest version of node.js for your operating system
  2. Download the latest version of git for your operating system
  3. Download and run the install script for Mac/Linux or Windows
  4. Open a terminal and browse to the nodegame/ folder
  5. Start the server with the command: node launcher.js
  6. Open two or more browser tabs pointing to localhost:8080/ultimatum
  7. Open a browser tab pointing to localhost:8080/ultimatum/monitor.htm
