Real-Time Social Experiments in the Browser

nodeGame is a free and open source javascript framework for online, multiplayer, real-time games and experiments in the browser.

News 27.06.2015: Release: nodeGame 1.0-beta

Released nodeGame 1.0-beta on github! Hurray!

Features:

  • Open source and open standard (HTML5)
  • 100% JavaScript code
  • Works on mobile devices and tablets
  • Zero installation for clients
  • Easy installation for server on Win/Mac/Linux
  • Modular framework (server / client / window / widgets / games)
  • Powerful API
  • Integrated NDDB JavaScript database
  • Server can run multiple games at the same time
  • Customizable waiting rooms for online games
  • Integrates smoothly with other libraries (e.g. jQuery, D3.js, etc.)
  • Integrates with external web services, such as Amazon Mechanical Turk

Get started

  1. Download the latest version of node.js for your operating system
  2. Download the latest version of git for your operating system
  3. Download and run the install script for Mac/Linux or Windows
  4. Open a terminal and browse to the nodegame/ folder
  5. Start the server with the command: node launcher.js
  6. Open two or more browser tabs pointing to localhost:8080/ultimatum
  7. Open a browser tab pointing to localhost:8080/ultimatum/monitor
