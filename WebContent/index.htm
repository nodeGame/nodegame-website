>nodeGame: Web Experiments in the Browser

nodeGame is a free, open source, event-driven javascript framework for on line, multiplayer games in the browser.

An official release will be available soon. For the moment the development branches are available on GitHub:
  • https://github.com/nodeGame/
  • https://github.com/shakty/nodeGame/
A brief documentation is included in the development releases.
