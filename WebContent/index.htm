Online, Real-Time, Synchronous Experiments in the Browser

nodeGame 3.2 is a free and open source javascript framework for online, multiplayer, real-time games and experiments in the browser.

_ Download Installer:

The latest stable version of nodeGame is 3.2. (release date 10/23/2016)
  • Stable:                Mac/Linux    or    Windows
  • Development:   Mac/Linux    or    Windows
For detailed instructions see the Get Started section, or the related wiki page.

_ Features:

  • Open source and open standard (HTML5)
  • 100% JavaScript code
  • Works on mobile devices and tablets
  • Zero installation for clients
  • Easy installation for server on Win/Mac/Linux
  • Modular framework (server / client / window / widgets / games)
  • Powerful API
  • Integrated NDDB JavaScript database
  • Server can run multiple games at the same time
  • Customizable waiting rooms for online games
  • Bots and phantoms
  • Templates and internationalization
  • Integrates smoothly with other libraries (e.g. jQuery, D3.js, etc.)
  • Integrates with external web services, such as Amazon Mechanical Turk
