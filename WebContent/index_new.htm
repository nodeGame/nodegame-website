nodeGame 4.0 alpha

Online Real-Time Synchronous Experiments.

Free and open source JavaScript framework for online, multiplayer, real-time games and behavioral experiments in the browser.

Ã—

Features

  • Open source and open standard (HTML5)
  • 100% JavaScript code
  • Works on mobile devices and tablets
  • Zero installation for clients
  • Easy installation for server on Win/Mac/Linux
  • Modular framework (server / client / window / widgets / games)
  • Matching algorithms for players
  • Integrated NDDB JavaScript database
  • Server can run multiple games at the same time
  • Customizable waiting rooms
  • Bots and "phantoms" to simulate human players
  • Templates and internationalization
  • Integrates smoothly with other libraries (e.g. jQuery, D3.js, etc.)
  • Integrates with external web services, such as Amazon Mechanical Turk
  • ...and more! See the documentation.
